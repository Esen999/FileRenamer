# File Renamer

Simple Python tool that renames all files in a folder using a new name pattern.

## Features
- Renames every file in the selected directory
- Adds incremental numbering (e.g., image_1.jpg, image_2.jpg)
- Works with any file type
- Helps organize messy folders

## How to Run
python file_renamer.py


## Example
Folder: photos  
New name: vacation  
Result:
- vacation_1.jpg  
- vacation_2.jpg  
- vacation_3.jpg
